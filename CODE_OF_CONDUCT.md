# Code of Conduct

This project adopts the [Contributor Covenant](https://www.contributor-covenant.org/version/2/1/code_of_conduct/) v2.1.

In short: be respectful and constructive. Harassment or abusive behavior is not tolerated.
Report conduct concerns to the maintainer via the repository's contact channels or a private GitHub report.
